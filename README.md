# 🛠️ Wholesale Electronic Shop Desktop Application  

---

## 📌 **Overview**

Welcome to the **Wholesale Electronic Shop**! 🚀  
This is a desktop application built using **Java** for the frontend and **MySQL** as the database for backend operations. The app is designed to manage wholesale electronic inventory efficiently.

---

## 🛠️ **Technologies Used**

- **Frontend**: Java (Swing/JavaFX for Desktop UI)  
- **Backend**: MySQL for database operations  
- **Database**: MySQL  
- **Development Environment**: NeBeans

---

## 💡 **Features**

✅ **Inventory Management**: Add, remove, and edit stock data.  
✅ **Customer Orders**: Manage orders from customers efficiently.  
✅ **Search Options**: Search by product name, category, or other criteria.  
✅ **Login Authentication**: Secure login system for different user access permissions.  
✅ **Reports Generation**: Generate stock and sales reports.
